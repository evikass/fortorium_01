Установка

Запустите студию на своём компьютере

💻

Требования

Node.js 18+, Bun runtime, Git

🚀 Быстрый старт

# Клонирование репозитория
git clone https://github.com/evikass/fortorium_01.git
cd fortorium_01

# Установка зависимостей
bun install

# Инициализация базы данных
bun run db:push

# Запуск development сервера
bun run dev

🔧 Интеграция с Blender

Для 3D сцен создайте файл blender_server.py на домашнем компьютере:

# blender_server.py
from http.server import HTTPServer, BaseHTTPRequestHandler
import bpy
import json

class BlenderHandler(BaseHTTPRequestHandler):
    def do_POST(self):
        content_length = int(self.headers['Content-Length'])
        data = json.loads(self.rfile.read(content_length))
        exec(data['script'], {'bpy': bpy})
        self.send_response(200)

# Запуск:
blender -b -P blender_server.py
