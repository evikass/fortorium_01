# AI Animation Studio - Мультиагентная Анимационная Студия

## Обзор

AI Animation Studio - это полноценная система для создания мультфильмов с использованием искусственного интеллекта. Система состоит из нескольких специализированных агентов, которые работают вместе автономно.

## Агенты

### 1. Продюсер (Producer)
- **Роль**: Координатор всего процесса
- **Функции**:
  - Планирование проекта
  - Создание задач для других агентов
  - Контроль качества
  - Координация пайплайна

### 2. Сценарист (Writer)
- **Роль**: Создание контента
- **Функции**:
  - Написание сценариев
  - Диалоги персонажей
  - Описание сцен
  - Адаптация под стиль

### 3. Художник (Artist)
- **Роль**: Визуальное оформление
- **Функции**:
  - Концепт-арты персонажей
  - Раскадровки (storyboard)
  - Фоны и окружение
  - Генерация изображений через AI

### 4. Аниматор (Animator)
- **Роль**: Оживление персонажей
- **Функции**:
  - Создание анимации из статичных кадров
  - Синхронизация губ
  - Движение камеры

### 5. Озвучка (Voice)
- **Роль**: Звуковое оформление
- **Функции**:
  - Генерация голосов персонажей
  - Фоновая музыка
  - Звуковые эффекты

### 6. Монтажёр (Editor)
- **Роль**: Сборка финального ролика
- **Функции**:
  - Монтаж видео
  - Наложение аудио
  - Добавление субтитров
  - Экспорт

### 7. Blender (3D)
- **Роль**: 3D контент
- **Функции**:
  - Создание 3D сцен
  - Настройка освещения
  - Анимация камеры
  - Рендеринг

## API Endpoints

### Проекты
- `GET /api/projects` - Получить все проекты
- `POST /api/projects` - Создать новый проект

### Задачи
- `GET /api/tasks` - Получить задачи
- `POST /api/tasks` - Создать и выполнить задачу
- `PUT /api/tasks` - Выполнить отложенную задачу

### Blender
- `GET /api/blender` - Получить подключения
- `POST /api/blender` - Создать/проверить подключение
- `PUT /api/blender` - Обновить статус
- `DELETE /api/blender` - Удалить подключение

## Интеграция с Blender

### Настройка сервера на домашнем компьютере

1. Создайте файл `blender_server.py`:

```python
from http.server import HTTPServer, BaseHTTPRequestHandler
import bpy
import json

class BlenderHandler(BaseHTTPRequestHandler):
    def do_POST(self):
        content_length = int(self.headers['Content-Length'])
        data = json.loads(self.rfile.read(content_length))
        
        # Выполняем скрипт в Blender
        exec(data['script'], {'bpy': bpy})
        
        self.send_response(200)
        self.send_header('Content-type', 'application/json')
        self.end_headers()
        self.wfile.write(b'{"success": true}')

# Запуск на порту 9876
server = HTTPServer(('0.0.0.0', 9876), BlenderHandler)
server.serve_forever()
```

2. Запустите сервер:
```bash
blender -b -P blender_server.py
```

3. Добавьте подключение в интерфейсе студии:
   - IP адрес вашего компьютера
   - Порт 9876
   - API ключ (опционально)

## Технологии

- **Frontend**: Next.js 16, React, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Prisma ORM, SQLite
- **AI**: z-ai-web-dev-sdk (GPT-4, DALL-E, TTS)
- **WebSocket**: Bun native WebSocket для Blender сервиса

## Запуск

```bash
# Установка зависимостей
bun install

# Инициализация базы данных
bun run db:push

# Запуск development сервера
bun run dev

# Запуск Blender сервиса
cd mini-services/blender-service && bun run dev
```

## Расширение

Для добавления нового агента:

1. Создайте файл в `src/lib/agents/your-agent.ts`
2. Наследуйте от `BaseAgent`
3. Реализуйте метод `execute()`
4. Экспортируйте в `index.ts`
5. Добавьте в UI на страницу агентов
